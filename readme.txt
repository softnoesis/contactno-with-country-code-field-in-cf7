Add phone number with country code & country flag field in contact form 7.

== Description ==
Conatct No With Country Code Field In Contact 7 Form plugin is **an add-on for Contact Form 7** plugin. This plugin **add two new form tag fields** that is **Country list** (form-tag: country drop-down) and **Country Phone extensions list** (form-tag: phone number) in Contact form 7.

Conatct No With Country Code Field In Contact 7 Form helps you in creating a country drop-down list with country flags. The tag field will automatically add countries name in standard drop-down field of contact form 7.

How to add the fields in the contact form 7 
1. Once you have installed activated the Phone Number With Country Code Field In Contact 7 Form plugin.
2. Add the form-tag  "country code" and  "phone number"  to your form and save the changes.

Requirments:
* Contact form 7 must be active plugin.

== Installation ==

1. Upload the entire `contactno-with-country-code-field-in-cf7` folder to the `/wp-content/plugins/` directory.
2. Kindly make sure 'contact form 7' plugin active before activate this plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Setup default country settings from contact >> PCF settings.

You will find two new fields type in your contact form 7 field list.

== Frequently Asked Questions ==

= How to set default selected country? =

Enter country 2 character ISO code. Default country will selected default country in dropdown. Contact >> PCF Settings. Example: US,IN,CA,GB. Contact >> PCF Settings

= How to enable Auto Country Select checkbox? =

Enabling this checkbox will automatically select the country dropdown based on your computers IP address. If a default country is not already selected, the dropdown will reflect the country associated with your IP address.Contact >> PCF Settings


= How disable International dial codes (+)? =

Enable option for dial code disable form plugin settings. Contact >> PCF Settings.
